SoniMinimap v0.1

Allows plotting of arbitrary points on Sonic Frontiers maps. Expected use case is exporting a CSV from Portal Gear and then copy/pasting the coordinates here to visualize them.

Usage: Input one xyz coordinate per line and click "load points"

I put very little thought / effort into this, so it may not work as you expect, or at all. Source code is available.
